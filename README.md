### Introduction

The project is based on Online Ecommerce Store where user can visit and buy their favourite products, also the product uses proper cart system.

### Overview

An ecommerce system in which we can see products like a shop, can view the product information and can add the items to the cart for buying purpose.

### Steps for Installation

1. Extract the zip file to any folder
2. Place the extracted folder to the web root
3. Visit your webroot/project_directory (e.g. localhost/eshop)
